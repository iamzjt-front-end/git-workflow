# Git Workflow 文档

这是 Git Workflow 的完整文档站点，使用 VitePress 构建。

## 📚 文档结构

### 指南 (`/guide/`)
- **介绍** - 项目介绍和设计理念
- **快速开始** - 三步开始使用
- **安装** - 详细安装说明
- **基础用法** - 掌握日常操作
- **AI 智能提交** - 深入了解 AI commit 功能
- **分支管理** - 高级分支管理技巧
- **Tag 管理** - 版本标签的高级用法
- **Stash 管理** - 高效管理代码暂存
- **版本发布管理** - 自动化版本发布流程
- **最佳实践** - 学习高效的工作流程

### 命令参考 (`/commands/`)
- **概览** - 所有命令的概览
- **交互式菜单** - 主菜单功能详解
- **分支命令** - 分支管理命令详解
- **提交命令** - 提交相关命令详解

### 配置文档 (`/config/`)
- **配置概览** - 配置系统介绍
- **配置文件详解** - 所有配置项的详细说明

## 🚀 本地开发

### 启动开发服务器

```bash
npm run docs:dev
```

访问：http://localhost:5173/git-workflow/

### 构建文档

```bash
npm run docs:build
```

### 预览构建结果

```bash
npm run docs:preview
```
